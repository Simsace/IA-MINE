Cadre juridique Les conventions d'établissement conclues entre le Gouvernement de la République du Mali et les sociétés minières opérant au Mali, en prospection ou en production, ont été signées, pour la plupart, sous l'empire des anciens codes miniers avant ladoption de celui de 2019. En vertu du principe de stabilité du régime juridique et fiscal, prévu dans ces différents codes, il s'avère qu'il subsiste donc cinq régimes au Mali : celui du Code de 1970, celui du Code de 1991, celui du Code de 1999, celui du Code de 2012 et celui du Code de 2019 avec la possibilité pour les sociétés minières d'opter pour le régime adopté dans le Code de 2019.
